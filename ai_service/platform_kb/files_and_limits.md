## Supported Upload Types

Supported formats: PDF (.pdf), Word (.docx), PowerPoint (.pptx), Plain text (.txt). Upload locations: Quiz Creator (File Upload tab), Flashcard Creator (File Upload tab), AI Tutor (clip icon), Materials Library (PDF only).

## File Size Limits

Maximum file size: 10MB for all uploads. If too large: split the document, compress the PDF, or paste text directly instead.

## Text Length Limits

Minimum 30 characters, maximum 50,000 characters for quiz and flashcard generation. Custom instructions: max 1,500 characters. Quiz: up to 25 questions total. Flashcards: 1–25 per generation batch. Flashcard question max 2,000 chars, answer max 4,000 chars.

## Request Timeouts

AI requests can time out under heavy load or for very large inputs. If this happens: retry, reduce input size, use fewer questions, or try again later.

## Model Context Limits

AI models have context window limits. Very long documents may be trimmed. For best results, focus on relevant sections and ask specific questions rather than pasting entire textbooks.

## Rate Limiting

Login, signup, quiz generation, and flashcard generation are rate-limited to prevent abuse. If rate limited, wait a few minutes before retrying.

## Best Practices For File Uploads

Keep files under 10MB. Use well-formatted documents for better text extraction. Remove irrelevant pages. Choose the appropriate difficulty level. Use specific custom prompts to guide generation.

## Common Error Messages

- File too large (max 10MB): reduce file size
- Failed to extract text: file may be image-based or corrupted
- Unsupported file type: use PDF, DOCX, PPTX, or TXT
- Please enter at least 30 characters: add more content
- Text is too long (max 50,000): shorten the input
- Generation failed: AI service temporarily unavailable, retry

## Browser And Device Support

Supported browsers: Chrome (recommended), Firefox, Safari, Edge. Works on desktop, laptop, tablet, and mobile. If something isn't working: clear cache, try a different browser, or check your internet connection.

## YouTube Quiz Input

In Quiz Creator, select the YouTube tab, paste a YouTube URL, and click Load Transcript. Captions must be available on the video. After the transcript loads, configure settings and generate. Supported URL formats: youtube.com/watch?v=..., youtu.be/..., youtube.com/shorts/...
